PlateCodes

 Repository (Mac/iOS)
Repository (Linux)

PlateCodes is an application that allows you to determine where an automobile obtained its license plate based on information on that plate. For example, all license plates for vehicles belonging to the embassy of Switzerland in Beijing, China begin with the number 213. If you see an Italian car with PI in the lower right hand corner, it means the license plate was issued by Pisa. See the README file for a list of supported search spaces.

PlateCodes is available for Mac and iOS. It's also available as a snap for Linux.

Tanks

 Repository

Tanks is a strategy game based on an online Flash game of the same name. It can be easily found online simply by searching 'tanks flash game' or similar queries. The game has some bugs regarding shields and is no longer on the homepage of the original creators. My goal is to create my own version of this game and offer new mechanics and features not available in the original game. See the README file for more details.

Tanks is available for Mac, but I'm writing the code to be as modular as possible. Porting the project to iOS should be easy, so that might be available later.

SockTalk

 Repository

SockTalk is a command line socket-based IM program. One machine runs a server program and other users connect via the client program. Each user can specify a username and the chat can be hosted on any open port. Commands are used to disconnect, display connected users, etc. More details are available in the README and also by running /help while connected in either program.

The SockTalk protocol is the same as the protocol used in the Java project on which SockTalk is based. SockTalk is written in C++ and should be compatible with any desktop platform. The program has been run on Ubuntu 16.04 LTS and Mac. If it turns out it uses a platform specific library or feature, please let me know by opening an issue.

Mancala

 Repository

Mancala is a strategy game in which pebbles are moved around a board. Full gameplay rules can be found in the man page. This is a command line version of that game for low performance computers or if you want to play while in a Terminal without starting X. The computer player prioritizes these moves, in this order:

  • Captures
  • Extra turns
  • Preventing the opponent from capturing pebbles
  • Preventing the opponent from taking extra turns
  • Picking the pocket farthest from the goal
This does not appear to be an optimal algorithm for winning at Mancala. If you can do better, feel free to fork the repository and work on it yourself. Make a pull request and I'll merge if the algorithm works.

Mancala is written in C and should be compilable on any platform.

Crystal Ball

 Repository

Crystal Ball randomly answers your yes/no questions in any of five languages: English, Italian, Simplified Chinese, German, and French. If you're ever unsure about something or want to pick from several options, Crystal Ball can help. DISCLAIMER: Please don't use Crystal Ball for any serious decision making. Answers are chosen pseudorandomly and should definitely not be used for important choices. Please consult a human being, preferably one with expertise in the relevant field, if you need to make such a decision and are unsure.

Crystal Ball is available for iOS (Mac version coming soon). If you're interested in the Mac version, check the repository regularly. I might add the Mac version and forget to update this page.

Project Toohak

 Repository

Kahoot! is an online quiz game platform. One user starts a quiz (hosted on the Kahoot's servers) and other players connect to the game via a web or mobile app. Questions and answers appear on the host user's computer and the players answer the multiple choice questions by clicking the appropriate answer on their devices. Project Toohak (Kahoot backwards) is a free, open source, socket- and LAN-based version of this game with the addition of some extra features. These can be found in the README. Its purpose is to serve as a suitable replacement for Kahoot! when network latency makes the game unfair to players with a bad connection by being playable on the local network.

Toohak is written in Java, so it can be run on any Java-compatible platform.

Arc Music Player

 Repository (Mac)
Repository (Linux)

The Arc Music Player is a lightweight, simple music player. It doesn't create libraries or sort your music and only has the absolute basics of a music player. This makes it small and easy to use while still providing most of the functionality required of a music player. See the README file for more details on available features. This app is useful when you have all your music in one place and don't want to copy it so that more than one application can access it. For example, if you have an iTunes library that you want to access on a separate partition (e.g. one running a different version of Mac), you can use the Arc Music Player instead of having two separate iTunes libraries for each version of Mac.

Arc Music Player is available for Mac and Linux. As of release 1.0, the two do not have compatible save formats. This may change in future releases. The Linux version is available as a snap package.
