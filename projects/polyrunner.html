Repository

Polyrunner was created for the 36th Weekly Game Jam on itch.io. The theme was Twins. Polyrunner is a procedurally generated infinite runner. You can make copies of yourself to overcome obstacles. See the README for full gameplay details. You can download the releases from GitHub or from itch.io.

Polyrunner was created using the Unity game engine. Platform availability:

  • Mac
  • Linux
  • Windows
