The Game of the Amazons is a board game in a similar family to go and chess. It was invented in 1988 by Walter Zamkauskas in Argentina. This project was inspired by this Numberphile video and consists of a backend library and command line frontend written in C, as well as graphical frontends for Mac and Ubuntu Touch. Refer to the READMEs, the YouTube video, or the Wikipedia page for details regarding gameplay.

The backend library is written in C and should be compatible with any platform. The frontend can also be run in the command line on any platform. Graphical frontends are platform specific.

Platform availability:

  • Rust frontend for all desktop platforms
  • Cocoa frontend for Mac (no longer maintained)
  • Ubuntu Touch
