The Arc Utilities Library is a set of scripts designed to automate certain tasks. See the beginning of each individual script for a shebang, information about script type and platform compatibility, and copyright and licensing information.

Scripting languages used in this library:

  • AppleScript (Mac only)
  • BASh (Any platform)
  • Perl (Any platform)
  • AWK (Any platform)

BASh and Perl scripts can, obviously, be run on any machine that has BASh/Perl. AppleScript is only available on Mac (unless there's a Windows interpreter I'm unaware of). AWK and BASh should be installed by default on Mac and Linux. Perl is installed by default on Mac.
