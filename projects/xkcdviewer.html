With xkcd Viewer, you can view xkcd comics on your Ubuntu Touch device. It can also be run on desktop environments. The viewer provides basic functionality, including:

  • Navigation to adjacent comics or to a random comic
  • Easy access to the explainxkcd page for the current comic
  • Pinch to zoom and showing of the comic title and alt text
  • A caching system for already-downloaded comics
The xkcd comics are created by Randall Munroe and available under CC BY-NC 2.5. See the repository's README for more details.

Platform availability:

  • Linux, Ubuntu Touch
