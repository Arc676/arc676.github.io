Find the Invisible * is based on a web game called Find the Invisible Cow. The project was created on request by a friend and offers users the ability to pick custom assets to use in the game. See the README file for more info on the gameplay. Also available from itch.io.

Platform availability:

  • Mac
An iOS version might be created in the future if a touchscreen analog for the game mechanics can be found.
