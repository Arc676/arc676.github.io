Repository

BrickBreaker, not to be confused with Breakout, is a game where you pop bubbles of the same color. Click a bubble to select all adjacent bubbles of the same color. Press any key to clear them. Clearing more bubbles at the same time gets you more points. The objective is to get as many points as possible before there are no more groups of adjacent bubbles of the same color. The game experience is highly customizable. BrickBreaker can be downloaded from GitHub or from itch.io.

Platform availability:

  • Mac
  • iOS (under development)
