Photonic Salvation is the randomly generated, first person game with an overly dramatic title where you have to turn off all the lights, located at random positions in the world, and return to bed in as little time as possible. The project uses Godot, an open source 2D and 3D game engine. All assets are available under Creative Commons licenses. This project was created for 7dfps and PROCJAM 2020.

Refer to the README for more details about the gameplay. Instructions are also available in-game.

Platform availability: Godot has export templates for Linux, Mac, and Windows. Binaries for all platforms are available from the repository releases and the itch.io page.
