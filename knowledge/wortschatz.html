German Vocabulary Sets

If you've read my About page, you'll know that I speak German. I also happen to have over 2000 terms on Quizlet. If you're also learning German and are interested in expanding your vocabulary or using Quizlet tools to practice/revise words, the sets are available to be used by anyone. You can access my German folder here.

If you don't want to use Quizlet, you can also download the sets as text. These files are unlikely to be updated, as I no longer update these Quizlet sets. Each line contains a new term with the format German term//English definition, for example: abfahren//to drive away. The files are sorted alphabetically by German term.
