Python Lessons

A while ago I wrote up a syllabus and some lesson materials for teaching Python on Raspberry Pis. This lesson content is in no way a comprehensive guide to Python, but does cover a good amount of what you should know about programming in Python. It also includes some basic knowledge about using BASh. Not all syllabus points have an associated task file; some task files cover multiple spec points. Not all task files include an actual task and therefore don't have an associated solution. If you find mistakes, please open a new issue.
